第四章

办公自动化基础

Word

用高版本打开低版本文档时，高版本的一些功能在低版本中是没有的，为了适应低版本，Word会使用“兼容模式”。

2023年12月7日 P131-135

在Word中，进行拖放式文字编辑时，要复制选定内容，则在拖动时要按住“Ctrl”键。

按“Ctrl+H”组合键可打开“替换”选项卡。

按“Ctrl+G”组合键可打开“定位”选项卡。

使用通配符查找时，如果要查找已被定义为通配符的字符，则要在该字符前键入反斜杠（\）。

2023年12月8日 P143-148

在Word中，大纲级别用于为文档中的段落指定等级结构，分为正文文本和1级至9级，正文文本级别最低

对具有多级项目的文档可以使用多级列表，单击“段落”功能区上的“多级列表”即可为段落应用多级列表。按“Tab”键可降低项目级别，“Shift+Tab”组合键可升级。

2023年12月8日 P148-153

在Word中，如果要指定精确的页边距值，在拖动边界的同时按住“Alt”健，标尺上会显示页边距值。

如果要为文档中不同章节设置不同的页眉页脚，可以使用分节符后在“页眉和页脚工具”中，禁用“链接到前一条”。

2023年12月8日 P153-157

Word内置的字符样式包括“强调”、“不明显强调”和“明显强调”。

Word内置的段落样式包括“普通”和“列出段落”。

启动Word时，新建空白文档的默认样式为“正文”样式。

绘制表格时，按住“Ctrl”键可以自动应用文字环绕格式。

在Word表格中，按“Tab”键移动到下一个单元格，按“Shift+Tab”组合键移动到上一个单元格。

当插入点位于表格的末尾时按“Tab”键，会自动添加一行表格。

2023年12月9日 P157-162

按“Ctrl+Enter”组合键可使表格分页。

在Word表格中，在拖动标尺标记的同时按住“Alt”键可显示行高、列宽的数值。

插入的新表格具有宽度为0.5磅的单实线默认边框。

2023年12月9日 P162-166

Word是以域的形式将计算结果插入选定单元格的。

如果计算公式中所引用的单元格内容发生了更改，可以选定该域，然后按“F9”键更新计算结果。

Word以链接和嵌入两种方式插入各种对象。

2023年12月10日 P166-172

对于多行公式，若要对齐公式中的运算符，可以在开始新行时按“Shift+Enter”组合键。

2023年12月10日 P172-177

快捷键“Alt+Shift+X”打开“标记索引项”对话框。

2023年12月11日 P177-186

Excel

Excel共1048576行、16384列，因此一个工作表有16384×1048576个单元格。

当前正在使用的单元格称为活动单元格。

2023年12月12日 P186-192

选定一个单元格后，按“Ctrl+箭头”组合键可以选定改行或该列的第一个或最后一个单元格。

按“Ctrl+Home”组合键可选择工作表中第一个单元格。

按“Shift+F8”组合键开启选定模式。

2023年12月12日 P192-195

若要在单元格中另起一行输入数据，可按“Alt+Enter”键输入一个换行符。

如果要在多个单元格中输入同一个数据，可选择要输入相同数据的单元格区域，输入数据后按下“Ctrl+Enter”确认输入。

在一个单元格中，最多可以存放32767个字符。

输入当天的日期可按“Ctrl+;”组合键。
输入当天的时间可按“Ctrl+Shift+;”组合键。

活动单元格边框右下角小方块称为“填充柄”。

2023年12月3日 P195-198

Excel包含四种类型的运算符：算术运算符、比较运算符、文本连接运算符和引用运算符。

如果要查看可用函数的列表，可单击某个单元格并按“Shift+F3”组合键。

对于选定的单元格地址，可按“F4”键在相对引用、绝对引用和混合引用之间进行切换。

2023年12月13日 P198-203

移动单元格时，拖动的同时按住“Ctrl”键，则进行复制操作。

按“Ctrl+1”快捷键打开“设置单元格格式”对话框

2023年12月14日 P203-208

快捷键“Ctrl+L”或“Ctrl+T”插入表格。

2023年12月15日 P212-217

在进行分类汇总之前，必须首先对即将要分类的列进行排序。

2023年12月16日 P224-230

PowerPoint

PowerPoint使用四种不同类型的动画效果：进入、退出、强调、动作路径。

2023年12月9日 P241-247

计算机的世界-课外拓展

端口

在网络技术中，端口(Port)大致有两种意思：

一是物理意义上的端口，比如，ADSL Modem、集线器、交换机、路由器用于连接其他网络设备的接口，如RJ-45端口、SC端口等等；
二是逻辑意义上的端口，一般是指TCP/IP协议中的端口，端口号的范围从0到65535，比如用于浏览网页服务的80端口，用于FTP服务的21端口等等。

端口含义

计算机“端口” 是英文port的义译，可以认为是计算机与外界通讯交流的出口。
其中硬件领域的端口又称接口，如：USB端口、串行端口等。
软件领域的端口一般指网络中面向连接服务和无连接服务的通信协议端口，是一种抽象的软件结构，包括一些数据结构和I/O（基本输入输出）缓冲区。
面向连接服务TCP协议和无连接服务UDP协议使用16bits端口号来表示和区别网络中的不同应用程序，网络层协议IP使用特定的协议号（TCP 6，UDP 17）来表示和区别传输层协议。
任何TCP/IP实现所提供的服务都是1-1023之间的端口号，这些端口号由IANA 分配管理。
其中，低于255的端口号保留用于公共应用；
255到1023的端口号分配给各个公司，用于特殊应用；
对于高于1023的端口号，称为临时端口号，IANA未做规定。

常用的保留TCP端口号有：
HTTP 80，FTP 20/21，Telnet 23，SMTP 25，DNS 53等。
常用的保留UDP端口号有：
DNS 53，BootP 67（server）/ 68（client），TFTP 69，SNMP 161等。

每个TCP报文头部都包含源端口号（source port）和目的端口号（destination port），用于标识和区分源端设备和目的端设备的应用进程。
在TCP/IP协议栈中，源端口号和目的端口号分别与源IP地址和目的IP地址组成套接字（socket），唯一的确定一条TCP连接。
相对于TCP报文，UDP报文只有少量的字段：源端口号、目的端口号、长度、校验和等，各个字段功能和TCP报文相应字段一样。

端口分类

按端口号可分为3大类：

（1）公认端口（Well Known Ports）：从0到1023，它们紧密绑定于一些服务。通常这些端口的通讯明确表明了某种服务的协议。例如：80端口实际上总是HTTP通讯。
（2）注册端口（Registered Ports）：从1024到49151。它们松散地绑定于一些服务。也就是说有许多服务绑定于这些端口，这些端口同样用于许多其它目的。例如：许多系统处理动态端口从1024左右开始。
（3）动态和/或私有端口（Dynamic and/or Private Ports）：从49152到65535。理论上，不应为服务分配这些端口。实际上，机器通常从1024起分配动态端口。但也有例外：SUN的RPC端口从32768开始。

常用端口

21：FTP服务所开放的端口，用于上传、下载文件。
22：SSH端口，用于通过命令行模式远程连接Linux服务器或vps。
23：Telnet端口，用于Telnet远程登录服务器。
25：SMTP服务所开放的端口，用于发送邮件。
80：HTTP用于HTTP服务提供访问功能，例如，IIS、Apache、Nginx 等服务。
110：POP3用于POP3 协议，POP3 是电子邮件收发的协议。
143：IMAP用于IMAP（Internet Message Access Protocol）协议，IMAP 是用于电子邮件的接收的协议。
443：HTTPS 用于HTTPS服务提供访问功能。HTTPS 是一种能提供加密和通过安全端口传输的一种协议。
1433：SQL Server SQL Server的TCP 端口，用于供SQL Server对外提供服务。
1434：SQL Server SQL Server的UDP端口，用于返回SQL Server使用了哪个 TCP/IP 端口。
1521：Oracle通信端口，服务器上部署了Oracle SQL需要放行的端口。
3306：MySQL数据库对外提供服务的端口。
3389：远程桌面服务端口，可以通过这个端口远程连接服务器。
8080：代理端口,同80端口一样，8080 端口常用于WWW代理服务，实现网页浏览。
